== Description ==

### Fact Check for Yoast SEO: getting all your facts straight!

Fake news is a serious issue these days, but we’re here to help you out! Fact Check for Yoast SEO integrates perfectly with the Yoast SEO for WordPress plugin. It supplements the SEO analysis with extra checks that scan your content for trustworthiness and actual facts. The plugin also makes suggestions for increasing the believability of your content, so your readers can rest assured. Use our plugin and you’ll always be sure your facts have been checked!

NB. Fact Check for Yoast SEO only works in combination with [Yoast SEO](https://wordpress.org/plugins/wordpress-seo/).

### Available languages

Fact Check for Yoast SEO understands English and Dutch. Don't forget to set your site language to one of those languages.

### Bug reports

Bug reports for Fact Check for Yoast SEO are [welcomed on GitHub](https://github.com/Yoast/fact-check). Please note GitHub is not a support forum, and issues that aren’t properly qualified as bugs will be closed.

== Installation ==

1. Visit 'Plugins > Add New'
2. Search for 'Fact Check for Yoast SEO'
3. Activate Fact Check for Yoast SEO from your Plugins page.
